# PortfolioXP

A Windows XP desktop simulator serving as a personal portfolio website.

Forked from [Paul Jaguin's PortfolioXP](https://github.com/UnMugViolet/portfolio).

## Disclaimers

**This is an ongoing project and it is not done yet**.
If you encounter any bugs it will be really helpful if you could provide the origin of that bug, the browser you used and the device.

**This design was not made by me**.
All the icons, wallpaper and sounds belong to Windows corporation. The original web implementation was created by Paul Jaguin.

## Features

- Loading Screen & Login
- Internationalisation (English)
- Play Minesweeper
- Reach me by email
- See my projects
- Notepad
- My CV
- My Images
- Listen to my playlist
- Manage the volume in the website

## Usage

```sh
npm install
```

### Compile with Hot-Reload and Scss for Development

```sh
npm run dev:all
```

### Compile and Minify for Production

```sh
npm run build
```
